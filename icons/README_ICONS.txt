ÍCONES TEMPORÁRIOS - SHAMAR LENS BR

Os ícones PNG precisam ser gerados manualmente.

OPÇÕES:

1. ONLINE (mais fácil):
   - Acesse: https://cloudconvert.com/svg-to-png
   - Upload: icon.svg
   - Converta para PNG nos tamanhos:
     * 16x16 → salve como icon16.png
     * 48x48 → salve como icon48.png  
     * 128x128 → salve como icon128.png

2. PHOTOSHOP/GIMP:
   - Abra icon.svg
   - Export/Save As PNG
   - Tamanhos: 16x16, 48x48, 128x128

3. INKSCAPE (gratuito):
   - Abra icon.svg
   - File > Export PNG
   - Width: 16, 48, 128

POR ENQUANTO: Extension pode funcionar só com .svg
(Chrome aceita, mas .png é recomendado)
