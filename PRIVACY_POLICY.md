# Política de Privacidade - Shamar Lens BR

**Última atualização:** 04 de janeiro de 2026

## Resumo

O Shamar Lens BR **NÃO coleta, armazena, transmite ou compartilha** nenhum dado pessoal ou informação de navegação dos usuários.

## Processamento Local

- Toda análise de conteúdo é realizada **100% localmente** no navegador do usuário
- Nenhum dado é enviado para servidores externos
- Nenhuma informação é transmitida pela internet

## Dados Armazenados

A extensão armazena **apenas** preferências de configuração localmente:
- Velocidade da animação do badge
- Preferências visuais do usuário

Estes dados são salvos apenas no dispositivo do usuário através da API `chrome.storage.local` e nunca são transmitidos.

## Permissões Utilizadas

### activeTab
Permite ler o conteúdo da página ativa apenas quando o usuário está navegando em sites de notícias suportados, para realizar a análise de confiabilidade.

### storage
Permite salvar as preferências de configuração do usuário localmente no navegador.

### scripting
Permite injetar o badge visual e o card de análise na página após o processamento.

## Sites Acessados

A extensão acessa apenas o conteúdo de páginas nos seguintes domínios:
- g1.globo.com
- uol.com.br
- folha.uol.com.br
- estadao.com.br
- veja.abril.com.br
- exame.com
- bbc.com/portuguese
- cnnbrasil.com.br
- gazetadopovo.com.br
- oglobo.globo.com

O acesso é apenas para **leitura e análise** do texto da notícia. Nenhum dado é coletado ou transmitido.
